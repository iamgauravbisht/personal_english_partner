## Personal Spoken English Partner

Step 1: clone this repository:

```bash
git clone https://github.com/iamgauravbisht/personal_english_partner.git
```

Step 2: install dependencies:

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

Step 3: run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
